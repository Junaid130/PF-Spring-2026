### Decision Structures

* **Understand the Concept of Decision Structures**
* [ ] Grasp the importance of decision-making in programming.
* [ ] Control the flow of a program.


* **Learn Basic Decision-Making Statements**
* [ ] Familiarize with **if**, **if-else**, and **switch** statements in C.


* **Implement if and if-else Statements**
* [ ] Use **if** to execute code blocks based on conditions.
* [ ] Use **if-else** for multiple branches and different actions.


* **Use switch Statements for Multiple Choices**
* [ ] Use **switch** for scenarios with multiple cases.
* [ ] Improve code readability and organization.


* **Understand Relational and Logical Operators**
* [ ] Use relational operators: **==, !=, <, >, <=, >=**.
* [ ] Use logical operators: **&&, ||, !**.
